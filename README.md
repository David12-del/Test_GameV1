# Survival Game

Браузерная 2D multiplayer survival игра.

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Игра откроется в браузере по адресу http://localhost:3000

## Управление

- **WASD** - движение
- **Пробел** - атака/добыча ресурсов
- **E** - съесть еду

## Геймплей

### Биомы
- Зелёный - лес
- Жёлтый - пустыня
- Серый - каменная зона

### Ресурсы
- Деревья (зелёные треугольники) - 3 удара
- Камни (серые круги) - 5 ударов
- Руда (синие ромбы) - 8 ударов

### Выживание
- Следи за HP и голодом
- Убивай животных для еды
- Голод уменьшается со временем
- При голоде 0 теряешь HP

### Мобы
- Животные (жёлтые) - нейтральные, убегают
- Враждебные (красные) - атакуют игроков
- Боссы (тёмно-красные) - много HP, сильная атака

### PvP
- Можно атаковать ботов
- С убитых ботов падает еда

## Открытие мира
